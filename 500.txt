Errores frecuentes con git 
git add .
git commit -m ""
git push origin main

Se crea el .git ignore para evitar un bucle al intentar subir el entorno. 

Error: "ModuleNotFoundError: No module named X"
Se visualiza si se hizo la correcta instalacion: pip list
Si no: pip install nombre_paquete


Error: git push falla

git pull origin main
